experiment module¶

class athena.utils.experiment.Experiment(name, model=None, solver_cls=None, train_args=None, log_dir_parent=None)[source]¶

Bases: object

Bundles information regarding an experiment. An experiment is performed on a model, with a Solver and the arguments that have to be used to train the model.

Parameters

  • name (str) – The name of the experiment.

  • model (nn.Module, optional) – The model to experiment on. Defaults to None.

  • solver_cls (BaseSolver, optional) – The Solver class to use. Defaults to None.

  • train_args (Dict[str, Any], optional) – The arguments to be passed on to the Solver when training. Defaults to None.

  • log_dir_parent (str, optional) – Parent of the log directory. The directory to store tensorboard logs and checkpoints. The log directory’s name will be the name of the experiment. Defaults to None.

Runs the experiment. More specifically, calls the BaseSolver.train method and saves the History.

athena.utils.functions.plot_experiments(experiments, save_path=None, figsize=18, 13)[source]¶

Plots the metrics of the experiments given. Experiments that have recorded the same metric will have overlapping plots.

Parameters

  • experiments (Union[Experiments, List[Experiment]]) – The list of Experiments to plot

  • save_path (str, optional) – The path to save the plot. Defaults to None.

  • figsize (Tuple[int, int], optional) – The size of the plot. Defaults to (18, 13).

Raises

Exception – When there are no metrics to plot.

athena.utils.functions.plot_misclassified(number, experiment, data_loader, device, save_path=None, figsize=10, 15, cmap='gray_r')[source]¶

Plots the misclassified images.

Parameters

  • number (int) – The number of misclassified images to plot.

  • experiment (Experiment) – The experiment that should be used to get the misclassified images

  • data_loader (DataLoader) – The DataLoader of the input data

  • device (str) – A valid pytorch device string.

  • save_path (str, optional) – The path to save the plot. Defaults to None.

  • figsize (Tuple[int, int], optional) – The size of the plot. Defaults to (10, 15).

  • cmap (str, optional) – The cmap to use while plotting. Defaults to ‘gray_r’

Raises

Exception – When solver of type athena.solvers.classification_solver.ClassificationSolver is not being used, and when the number of misclassified images are less than the number of images being requested to plot.

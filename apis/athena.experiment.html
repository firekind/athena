class athena.experiment.Experiment(name, model, solver)[source]¶

Bases: object

Bundles information regarding an experiment. An experiment is performed on a model, with a Solver.

Parameters

  • name (str) – The name of the experiment.

  • model (nn.Module) – The model that the experiment is acting on.

  • solver (BaseSolver) – The solver the experiment is using.

run()[source]¶

Runs the experiment. More specifically, calls the BaseSolver.train method and saves the History.

class athena.experiment.Experiments(name, experiments, log_dir=None)[source]¶

Bases: object

Defines a list of experiments that has to be run one after the other.

Parameters

  • name (str) – The name of the list of experiments.

  • experiments (Dict) – The name-experiment key value pairs.

  • log_dir (str, optional) – The parent of the directory where the logs should be stored. the directory where the logs will be stored will be the name parameter.
